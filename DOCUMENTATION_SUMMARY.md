## Usage Instructions

### Local Development
```bash
# Install dependencies
pip install -r requirements-docs.txt

# Serve documentation locally
mkdocs serve

# Build static site
mkdocs build
```

### Deployment Options
- **GitHub Pages**: `mkdocs gh-deploy`
- **Netlify**: Connect to Git repository
- **Custom Server**: Deploy `site/` directory
- **Docker**: Container-based deployment
