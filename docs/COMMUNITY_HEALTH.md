## How to Use

### Reporting a Bug
1. Go to [Issues](https://github.com/GSejas/gacetachat/issues/new/choose)
2. Select "Bug Report"
3. Fill out the template
4. Submit

### Requesting a Feature
1. Check existing issues first
2. Go to [Issues](https://github.com/GSejas/gacetachat/issues/new/choose)
3. Select "Feature Request"
4. Explain use case and priority

### NGO Partnership Inquiry
1. Go to [Issues](https://github.com/GSejas/gacetachat/issues/new/choose)
2. Select "NGO Partnership Inquiry"
3. Fill out organization details
4. Or use web form: https://gacetachat.streamlit.app/ (faster)

### Submitting a Pull Request
1. Fork repository
2. Create feature branch
3. Make changes
4. Run tests: `pytest -v`
5. Open PR (template auto-fills)

### Reporting Security Issues
**Do NOT use public issues for security vulnerabilities.**
Email: security@gacetachat.cr
